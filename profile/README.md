### GitHub & GitLab commit convention
- **[Feat]** 새로운 기능 구현
- **[Add]** Feat 이외의 부수적인 코드 추가, 라이브러리 추가, 애셋(사진, 컬러) 추가
- **[Create]** 새로운 파일 생성
- **[Design]** UI 수정사항
- **[Fix]** 코드 수정, 내부 파일 수정, 주석
- **[Handle]** 버그, 오류 해결
- **[Delete]** 쓸모없는 코드 삭제
- **[Move]** 파일 이름/위치 변경
- **[Update]** 버전 업데이트
